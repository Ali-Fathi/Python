# Naming and Using Variable in Python

- Rules an guidelines to adhere to
    * Only letter, numbers and underscores
        * Can start with letter or underscore but not a number
            * e.g _name, name_1
    * No spaces in variable names but use underscores instead
        * e.g my_name
    * Avoid using python keywords
        * e.g print
    * variable should short and descriptive
    * make the letters obvious.
